1. install glad headers
2. install glfw package via package manager
3. compile static library `papyrus` in `src` folder
4. compile test program `papyrus-test` in `test` folder

how to work with cmake:
1. `mkdir build`
2. `cd build`
3. `cmake ..`
4. `make`
5. `sudo make install` (`src` folder) / `./papyrus-test` (`test` folder)
